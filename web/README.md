# The Auto-Mate - Automation Learning Platform

A modern web application built for **The Auto-Mate**, an online learning platform dedicated to automation testing and software skills. This project showcases a responsive, high-performance landing page with premium aesthetics.

## 🚀 Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Form Handling**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)

## ✨ Features

- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Modern UI/UX**: glassmorphism effects, gradient text, and smooth transitions.
- **Components**:
  - **Dynamic Hero Section**: With animated elements and gradient typography.
  - **Features Grid**: Showcasing unique selling points using Lucide icons.
  - **Courses Section**: Displaying available learning paths.
  - **Testimonials Carousel**: Interactive slider with snap scrolling logic.
  - **Stats Section**: Highlighting platform achievements.
  - **Contact Form**: Integrated UI for user inquiries.
  - **Blog Section**: Latest updates and articles.

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- pnpm (recommended) or npm/yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd automation-app
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Run the development server:**
   ```bash
   pnpm dev
   # or
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application.
